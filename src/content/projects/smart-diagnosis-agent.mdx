## 项目背景

线上问题排查传统依赖人工翻日志、读代码，链路长、耗时久。

## 做法

Agent 自主监测日志与告警，结合 CodeMap 代码索引定位问题代码并执行修复，形成「监测 → 定位 → 修复 → 验证」的全链路闭环，显著提升线上问题排查效率。

前端开发，参与诊断系统的前端与 Agent 流程建设。
