## 项目背景

广告主选号——为投放挑选合适的达人 / 账号——长期依赖人工，效率低、对经验门槛要求高。

## 做法

以对话为入口，Agent 编排内容理解、联网搜索与选号 Skill，自主走完「需求理解 → 资料收集 → 达人匹配 → 营销方案生成」的完整选号流程，把原本零散的人工操作收敛为一次对话。

## 我的角色

前端主导（Front-End Lead），负责选号系统的前后端开发与 Agent 交互体验。
